# 💰 CoreNaksh Finance

> A modern, production-ready personal finance application built with Flutter.

CoreNaksh Finance is an all-in-one financial toolkit designed to help users calculate, plan, and manage their finances with accuracy and simplicity.

---

# ✨ Features

## 📈 Investment Calculators

- ✅ SIP Calculator
- ✅ Fixed Deposit (FD) Calculator
- ✅ Recurring Deposit (RD) Calculator
- ✅ Public Provident Fund (PPF) Calculator

## 🏦 Loan Calculators

- ✅ EMI Calculator

## 🧾 Tax Calculators

- ✅ GST Calculator
- 🚧 Income Tax Calculator (In Progress)

## 📄 Reports

- PDF Export
- Share Results

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/corenaksh-pixel/financehub.git
```

## Go to Mobile App

```bash
cd financehub/apps/mobile
```

## Install Packages

```bash
flutter pub get
```

## Run

```bash
flutter run
```
